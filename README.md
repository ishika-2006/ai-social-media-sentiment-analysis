# Social Media Sentiment Analysis

## Project Overview

This project focuses on analyzing sentiments expressed in social media posts using Artificial Intelligence and Machine Learning techniques.

## Dataset

Sentiment140 Dataset

## Source

https://www.kaggle.com/datasets/kazanova/sentiment140

## Project Structure

### data

Contains raw datasets, cleaned datasets, and project documentation.

### src

Contains source code and Jupyter notebooks used in the project.

### reports

Contains reports and project analysis documents.

### deployment

Contains deployment-related resources and configurations.
